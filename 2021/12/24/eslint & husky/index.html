前置知识：git hooks 详解

基本用法

husky 对 git hooks 进行了封装，使我们能够更方便的使用 git hooks。最基础的用途就是保证 git commit 提示信息的格式。这里使用的 husky 版本是 7+。husky 做过一个 break change，因此如果使用的是低于 7.x 的版本，初始化命令的执行会有问题。
