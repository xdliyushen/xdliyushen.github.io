yaml 是什么？

yaml 的全称是 YAML Ain't a Markup Language 或者是 Yet Another Markup Language ，是一种可读性高、用来表示信息序列化的格式。可以将它理解为另一种 JSON，但它远比 JSON 要方便的多。yaml 常用于书写配置文件或元数据。

yaml 的语法

yaml 一般以三个相邻的 - 为开头，接着是要表达的数据，这些数据都以 key-value 的形式存在。
